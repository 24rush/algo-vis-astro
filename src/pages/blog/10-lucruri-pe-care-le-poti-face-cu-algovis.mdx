# Ce este Algovis?
Algovis este _widget-ul_ ce te ajută să faci primii pași în info. El îți arată cum funcționează cele mai uzuale construcții ale limbajului Javascript făcând pași prin secvențe de cod.

Mai jos găsești toate opțiunile disponibile pe interfața widget-ului:
<img src="../assets/imgs/algovis.png" class="img-box" />

# Ce pot face cu Algovis?

## 1. Înțelege ce sunt variabile de tip primitiv și obiect

<Arrow/>sari la lecțiile aferente: [Sintaxa unui limbaj de programare](./../notiuni_de_baza/sintaxa) și [Tipuri de date](./../notiuni_de_baza/tipuri-date)

<Algovis av_lazy>
```js
let x = 5; /*** creăm variabila x a cărei valore inițială este 5***/
x = 6; /*** modificăm conținutul variabilei x să fie acum 6***/
console.log(x); /*** afișăm conținutul variabilei***/

x = "Alin"; /*** modificăm tipul variabilei să conțină un text***/
console.log(x);

let nume = "Amalia"; /*** declarăm variabila nume cu valoarea Amalia de tip text.***/

let y = x; /*** declarăm variabila y ca fiind o copie a variabilei x ***/
let suma = x + y; /*** declarăm variabila suma ca fiind suma celor două variabile x și y ***/

let info_joc = {
         scor: 12412,
         jucator: "Ionut",
         ultim_nivel_jucat: 4
   };
console.log(info_joc.scor);
```
</Algovis>

## 5. Folosi structuri repetitive
<Arrow/>sari la lecția aferentă: [Structuri de control și repetitive](./../notiuni_de_baza/structuri-control)

<Algovis av_lazy>
```js
let repeta = prompt("Valoare repetitii");

for (let i = 1; i <= parseInt(repeta); i++) {
  console.log(i);
}

while (repeta > 1) {  
    console.log(repeta);
    repeta /= 2;
}
```
</Algovis>

## 6. Observa cum funcționează operațiile de bază pe vectori / stive / cozi
<Arrow/>sari la lecțiile aferente: [Vectori](./../notiuni_intermediare/vectori) sau [Stive / cozi](./../notiuni_intermediare/stiva_coada)

<Algovis av_lazy>
```js
let vector = ["A", "B", "C"];
for (let i = 0; i < vector.length; i++) {
    console.log(vector[i]);
}

let stack = [1, 2, 3, 4];

// Adaugam elementul 5
stack.push(5);

// Eliminam primul element
stack.pop(); // 5

let queue = [1, 2, 3, 4];

// Adaugam elementul 5
queue.push(5);

// Eliminam primul element
queue.shift(); // 5
```
</Algovis>

## 8. Întelege ce sunt referințele
<Algovis av_lazy>
```js
function SticlaApa(v, g) {
    this.volum = v;
    this.greutate = g;
}

let sticla1 = new SticlaApa(500, 700);
let sticla2 = sticla1;
sticla2.volum = 1000;

console.log(sticla1.volum);

function foo(a) {
    console.log(a);
    a.valoare = 6;
}

let numar = { 'valoare' : 123 };
foo(numar);

console.log(numar.valoare);

```
</Algovis>

## 10. Crea și vizualiza arbori
<Arrow/>sari la lecția aferentă: [Arbori
](./../notiuni_avansate/arbori)

<Algovis av_lazy>
```js
let gentree = new BinarySearchTree();
gentree.generate(8);

// Creaza un arbore manual
let tree = new BinaryTree();
let root = tree.createRoot(10);
let rootLeftChild = new BinaryTreeNode(5);
let rootRightChild = new BinaryTreeNode(15);

root.left = rootLeftChild;
root.right = rootRightChild;

let rootLeftLeftChild = new BinaryTreeNode(12);
root.left.left = rootLeftLeftChild;

```
</Algovis>
